# Card Game Suite

A modern, modular web application featuring classic card games: **War** and **Blackjack**.

## Features

*   **Multi-Game Architecture:** Easily switch between "War" and "Blackjack".
*   **Modern Visuals:**
    *   Programmatic SVG card graphics (high quality at any size).
    *   CSS 3D transforms for realistic card flips.
    *   Casino-style table aesthetics.
*   **Blackjack:**
    *   Full implementation of rules (Hit, Stand, Bust, Dealer Logic).
    *   Push handling and Ace calculation.
*   **War:**
    *   Classic rules including chained Wars.
    *   Automated gameplay animations.
*   **Audio:** Sound effects for dealing, flipping, and chips.
*   **Settings:** Toggle audio and adjust animation speed.

## Play Now
Open `index.html` in a modern web browser.
